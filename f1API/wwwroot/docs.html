Getting Started

Here's a quick guide to using the F1 API. It's packed with queries for you to explore. Right now, you can dig into info about two main areas:

  • Drivers
  • Teams

Whether you're looking for driver stats or team details, you'll find what you need. Just follow the steps here, and you're all set to tap into the world of F1 racing data.

Baseurl

http://localhost:5055/api/

The base url doesnt work by itself, you need to add a endpoint to it.

Drivers

There is a total of 0 drivers sorted by id.

Driver schema

Key Type Description
id int Id of the driver.
name string Name of the driver.
age int Age of the driver.
nationality string Nationality of the driver.
image string Image of the driver.

Get all drivers

You can access the list of drivers by using the /drivers endpoint.

GET http://localhost:5055/api/drivers

Get single driver

You can access a single driver by adding the /id of the driver as a parameter drivers/id.

GET http://localhost:5055/api/drivers/4

 {
  "id": 4,
  "name": "Fernando Alonso",
  "age": 42,
  "nationality": "Spain",
  "image": "alonso.avif"
 }

Get drivers by name

You can access the list of drivers containing inserted /name as a parameter /drivers/getbyname/name.

GET http://localhost:5055/api/drivers/getbyname/Lan

Teams

There is a total of 0 teams sorted by id.

Team schema

Key Type Description
id int Id of the team.
manufacturer string Manufacturer of the team.
driver1 string Name of driver 1.
driver2 string Name of driver 2.
imageCar string Image of the car.
imageLogo string Logo of the team.
